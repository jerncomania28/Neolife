efris.neolifeafrica.info

Introduction

commissions.neolifeafrica.info in it's simplest form is a system built to reward distributors within a business period. This reward is a culmination of sales volume bonus, leadership development bonus, network development bonus, consistency bonus and rebates. These various incentives/bonuses that are paid to distributors are generated on Exigo and passed to this system which does some deductions based on the local country regulatory policies to generate the final payouts that are sent to distributors. On a top level there are various functionality built into this system as listed under the uses below

Uses

  • Settings
  • Admin Fee
  • Adjustments
  • Commissions Run
  • Dashboard Overview

Features

  • Settings This is where we set up configurations such as the business period for running the commission, disabling and enabling sms pushout for successful commissions and finally enabling/disabling the process of commission upload.

  • Admin Fee This is another feature that enables us pass certain adminstrative fees to distributors based on their level and countries. We always look into the comAdminfees table that has an already defined values for this.

  • Adjustments This is where all the necesssary adjustments that are to be made on the commissions are applied. These adjustments could be money we owe the distributor or any deduction to be made to balance accounts accross board. Adjustments could be added as a bulk upload by uploading csv files or single entry. The fields needed to create an adjustment are the business period, NeolifeID and Description .

  • Dashboard Overview This is a single pane of glass window that shows summary of progress of commission run so far. These includes total rows in volumes, total rows processed from volumes, total adjustments, status, reports upload progress and upload points status.

  • Commision Run This is where the actual commisions are ran and the various payadvice are generated for the distributor. There are about eleven sub-proceess that needs to happen to achieve this commision run. They are listed below

    Run Commissions Summaries : Extracts commission details from Exigo and insert them into extended table

  • Run Commissions Volume Summaries : Generate consistency bonus and extract commission details from commisions summaries and insert into the extended table

  • Copy Adjustments : copy out created adjustments into payment details

  • Create Payout Details : Extract Bonus, retail and member profits, VAT , consistency bonus, withholding tax, admin fees, exchange costs and medical aids.

  • Extract Bank Details : Extract customers Bank account number and bank name from Exigo custom reports

  • Approve Final Payouts : Run Reports and verify final payouts then generate payout files

  • Create Commissions Points: Create commissions points for non Direct Deposits

  • Populate SSRS Reports : Populate SSRS reports and background tasks

  • Merge Business Profile : Merge all Business Profile SD and above

  • Populate Commisions Notification SMS : Populate commissions Notification SmS into the customservicescontext.smsmessages table.

  • Create and Upload External Reports : Create and upload all external Reports to Amazon s3
