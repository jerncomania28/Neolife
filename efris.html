efris.neolifeafrica.info

Introduction

This is a project that was requested by the Uganda revenue authority to enable Neolife remit tax collation for every product order real-time to the Uganda Revenue Authority. This is built as a microservice which runs unaided

Uses

  • GOODS UPLOAD
  • STOCK UPLOAD
  • ISSUE-INVOICE
  • ISSUE CREDIT NOTE

Features

  • GOODS UPLOAD

    This is the feature that enables Neolife push all the products that are sold to Efris. This is a reoccuring process as Neolife receive new products

  • STOCK-UPLOAD

    This is the feature that enables Neolife upload the quantity of each of these products to efris

  • ISSUE-INVOICE

    This is the feature that enables Neolife issue invoice as they sell products which is in turn reflected on Efris

  • ISSUE-CREDIT NOTE

    This is the feature that enables Neolife issue credit note for returned products which affects Efris stock position
