banks.neolifeafrica.info

Introduction

This is the web service api that powers all neolifeafrica's bank integration . This api enables banks and payment providers to integrate with Neolife enabling neolife distributors make payments for products purchased.

Various Banks/Payment Providers Integration

  • AirtelMoney
  • Ecobank
  • Interpay
  • Interswitch
  • MTNController
  • Mcommerce
  • Mpesa
  • NGenius
  • Paygate
  • Paystack
  • TigoPesa
  • USSD
  • Vodacom
  • WireCard
  • WirecardEFT

Features

  • AirtelMoney

    This is the payment integration that enables neolife distributors in uganda make payments on their mobile money

  • Ecobank

    This is the payment integration for payments made by neolife distributors that uses Ecobank in Ghana

  • Interpay

    This is the payment integration for payments on Interpay

  • Interswitch

    This is the payment integration for payment on Interswitch

  • MTNController

    This is the payment integration that enables neolife distributors make payment on MTN

  • Mcommerce

    This is the payment integration that enables payments on mcommerce which is powered by ogaranya

  • Mpesa

    This is the payment integration that enables Kenya distributors make payment

  • NGenius

    This is the payment integration that enables Neolife distributors in Namibia make payments

  • Paygate

    This is the payment integration that enables Neolife distributors in Togo make payments

  • Paystack

    This is the payment integration that enables Neolife distributors in Nigeria make payments

  • TigoPessa

    This is the payment integration that enables Neolife distributors in Tanzania make payments

  • USSD

    This is the payment integration that enables payments using USSD code

  • Vodacom

    This is the payment integration that enables Neolife distributors in Tanzania make payments

  • Wirecard

    This is the payment integration that enables South African distributors make payments

  • WirecardEFT

    This is the payment integration that enables South African distributors make payments through electronic funds transfer
