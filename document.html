office.neolifeafrica.info

Introduction

The office.neolifeafrica.info is a web solution that helps staff manage daily operations from a single pane of glass. This solutions introduces a user interface enabling a seamless access to custom solutions built to support the Exigo platform.

Uses

  • Deposits
  • Web Payments
  • Create Credit Notes
  • Account Number Lookup
  • Convention (ZA)
  • SMS Monitor
  • Bulk SMS
  • POS Invoice
  • Subscriptions
  • Reverse Commision Points
  • Auto Print
  • Distributors Report
  • Logout

Features

  • Deposits

    This feature is available as a pointer to the web reports on the deposits table Reports are organised into folders. All related reports are put in the same folder . The following is the list of folders available

  • web Payments

    This feature is available as pointer to the web payments reports.

  • Create Credit Notes

    This is a feature on office.neolifeafrica.info that allows neolife staff to create a credit note for distributors on payment. The details captured in this process includes the neolife id , customer phone , amount , source , deposit description and any other additional details .

  • Account Number Lookup

    This feature allows Neolife staff to look up account names of distributors to verify the authenticity of their account details. This is done using the account number and bank name of the distributor .

  • Convention (ZA)

    This feature was created for a convention in 2019 to enable pull ticket for orders

  • SMS Monitor

    This feature enables real-time monitoring of sms sent . it aggregates these records by country , sms sent today , pending and total sms

  • Bulk SMS

    This feature enables staff to format and send bulk sms to neolife distributors

  • POS Invoice

    This feature allows seamless printing of invoice using the order id of distributors

  • Subscriptions

    This feature enables seamless creation of new subscriptions

  • Reverse Commision Points

    This is used to reverse commission points for neolife distributors

  • Auto Print

    This feature enables you to automatically assign an email to receive an invoice that will be automatically printed .

  • Distributors Report

    This features enables staff view distributors report using specific neolife id and period

  • Logout

    This enables staff to logout once they are done with their operations.
