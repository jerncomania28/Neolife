api.neolifeafrica.info

Introduction

api.neolifeafrica.info is the web api that powers the office.neolifeafrica.info . it serves as a backend service for the office.neolife.africa

Uses

  • Create Credit Notes API
  • Account Number Lookup API
  • Convention(ZA) API
  • SMS Monitor API
  • Bulk SMS API
  • POS Invoice API
  • Subscriptions API
  • Reverse Commision Points API
  • Auto Print API
  • Distributors Report API
  • Logout API

Features

  • Create Credit Notes API

    This is the backend service that enables users to create credit notes on office.neolifeafrica.info

  • Account Number Lookup API

    This is the backend service that enables users to lookup account name of neolife distributors

  • Convention(ZA) API

    This is the backend service that enables neolife staff handle the conventions process on office.neolifeafrica.info

  • SMS Monitor API

    This is the backend service that powers the SMS monitoring .

  • Bulk SMS API

    This is the backend service that powers the bulk sms sending

  • POS Invoice API

    This is the backend service that powers the POS invoicing

  • Subscriptions API

    This is the backend servicec that powers the subscriptions

  • Reverse Commision Points API

    This is the backend service that powers reversing commision points

  • Auto Print API

    This is the backend service that powers auto printing

  • Distributors Reports API

    This is the backend service that powers pulling distributors reports from AWS S3

  • Logout API

    This is the backend service that enables logout

FAQ

How do i know when this service is up and running ?

Simply go to your browser and type api.neolifeafrica.info and see if it opens up a content

What happens when api.neolifeafrica.info is down ?

You won't be able to carry out most transaction on office.neolifeafrica.info You won't be able to see any content when you run api.neolifeafrica.info
